# Shivam Glass & Aluminium Works Website

Simple static showroom website for the family business.

## Files

- `index.html` - page structure and content
- `style.css` - responsive styling and layout
- `script.js` - contact links, form-to-WhatsApp flow, and rendered cards

## Before going live

1. Replace the placeholder phone and WhatsApp number in `script.js`.
2. Replace the placeholder Google Maps link in `script.js`.
3. Swap the gallery placeholder cards with real project photos.
4. Replace the starter testimonials with actual customer reviews.

## Preview

Open `index.html` in a browser.
